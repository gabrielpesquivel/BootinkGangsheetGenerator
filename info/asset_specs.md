# COLOURS:
- Black text (221f1f)
- Clear background

# FONT:
- INDUSTRY (ULTRA)

# SIZES (all are heights):
- Flags: 6mm 
- Symbols: 10mm
- Initials + Numbers: 5mm
- Dates + Words over 3 characters: 4mm

# GRID:
- Grid line every 25mm (Anything over 3 characters = 2 grid squares, over 10 characters = 3 grid squares)

# OFFSET:
- 0.5 mm
- Fill inwards from bubble 
- if white words, black fill
- if black words, white fill
- opacity 3% 

# PAGE:
- Transparent
